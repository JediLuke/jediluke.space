Developer.mozilla.org defines HTML5 as:

"a markup language used for structuring and presenting content on the World Wide Web. It is the latest evolution of the standard that defines HTML. The term represents two different concepts. It is a new version of the language HTML, with new elements, attributes, and behaviors, and a larger set of technologies that allows the building of more diverse and powerful Web sites and applications."

Will you be surprised that a simple page like this below contains a few HTML elements and tags so you can be able to view it on your browser?

Wikipedia defines the HTML element as:

"a type of HTML document component, one of several types of HTML nodes. An HTML document is composed of a tree of simple HTML nodes, such as text nodes, and HTML elements, which add semantics and formatting to parts of the document. Each element can have HTML attributes specified."

So, in a simple web page or HTML document, it consists of multiple HTML elements, defined by an opening tag, content, and a closing tag.

To display "Hello World!" in the the first line of your web browser, you need to have these codes:

<!DOCTYPE html> <html> <head> <title>Hello World<title> <head> <body> <h1>Hello World!<h1> <body> <html>

See the definition and usage of each tag below:


The <!DOCTYPE html> - each HTML document starts with this Document Type Declaration, which tells the web browsers the version of the markup language in which a web page is written.

The <html> tag - represents the "root" of the HTML document. All the HTML elements and tags will be placed inside these tags so that our page will be displayed by web browsers correctly.

The <head> tag - is placed between the <html> tag and the <body> tag, which serves as a container of the metadata of the HTML document. Metadata defines the document title, character set, styles, scripts, and other meta information that are not displayed on the browser.

The <title> tag - specifies the title of the document that is shown in a browser's title bar or page's tab and will be visible in search engine results pages as a clickable headline for a given result. Also, it's one of the most important elements of usability, SEO, and social sharing.

The <body> tag - defines the document's body, which contains all the contents of an HTML document, such as headings, paragraphs, images, videos, hyperlinks, tables, lists, etc., that are visible in your web browsers.

The <p> tag - defines the blocks of paragraph text. The browsers always add margin automatically before and after each <p> tags or paragraphs.

As you can see, each element has an opening and closing tags. There are also elements with no closing tags, known as empty tags such as <br>, <img>, <hr>, <link>, <meta>, <source>etc.
